Реализовано: Поиск города, Просмотр текущей погоды и прогноза на 5 дней, добавление/удаление города из избранных, определение города в котором находится пользователь.

Технологии
---
+ SwiftUI
+ MVVM
+ CoreData
+ UserDefaults
+ CoreLocation
+ Alamofire
+ Realm
